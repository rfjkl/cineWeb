# CineWeb — Projeto completo

## Como rodar

1. Instale dependências:
```bash
npm install
```

2. Inicie o json-server:
```bash
npm run json-server
```
API em http://localhost:3000

3. Inicie o frontend:
```bash
npm run dev
```
Acesse a aplicação em http://localhost:5173

## Observações
- Validações são feitas com Zod.
- Rotas: /filmes, /salas, /sessoes, e /sessoes/vender/:id
